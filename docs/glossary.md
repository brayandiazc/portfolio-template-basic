# Glosario

Vocabulario compartido del dominio y del proyecto. Mantén las definiciones cortas
y sin ambigüedad para que todo el equipo use los términos de la misma forma.

| Término     | Definición                    |
| ----------- | ----------------------------- |
| Proyectos            | Lista **curada a mano** en `assets/js/projects.js` (título, descripción, enlaces); no depende de GitHub. |
| Repositorios         | Repos públicos del usuario **con estrellas**, traídos de la API de GitHub y ordenados de más a menos.    |
| Tema (`data-theme`)  | Modo visual claro u oscuro, controlado por el atributo `data-theme` en `<html>`.               |

> Convención: ordena los términos alfabéticamente y enlaza al documento donde se
> detalla cada concepto cuando aplique.
